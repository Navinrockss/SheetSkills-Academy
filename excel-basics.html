Lesson 1 — First steps: open, save, enter data

Goal: feel confident opening Excel or Google Sheets, entering data, saving, and naming sheets — the basics that make everything else possible.

Step-by-step (do this now)

  1. Open Excel or Google Sheets and create a new blank workbook.
  2. Rename the first sheet: double-click the sheet tab and type Sales.
  3. In row 1, type headers: Date, Product, Qty, Unit Price, Notes.
  4. Enter a few rows of sample data (3–10 rows). Click a cell and type — press Enter to go down the column.
  5. Save the file with a clear name: Sales_Jan_2025.xlsx (or simply use Google Sheets which auto-saves).
Why this matters: clear sheet names and headers help you find and filter information quickly later.

Lesson 2 — Formatting for clarity

Goal: make spreadsheets readable so colleagues or managers instantly understand the data.

Actions to perform

  • Make headers bold and slightly larger than data rows.
  • Align numbers to the right and text to the left.
  • Format price and amounts as currency (use the Number / Currency format).
  • Apply a light fill color to header row for contrast.
  • Wrap long text in description/notes cells so everything is visible.

Practice checklist

Common mistakes:
  • Storing numbers as text: when copied from other places, numbers may be text — use "Convert to Number" or retype.
  • Different date formats: keep dates consistent (e.g., DD-MM-YYYY or YYYY-MM-DD).

Lesson 3 — Convert data to a Table & use Filters

Goal: quickly show only the rows you need (e.g., one product, one date range) and keep formatting when you add rows.

Steps (Excel)

  1. Select any cell in your data range and press Ctrl + T (or Insert → Table).
  2. Ensure "My table has headers" is checked and click OK.
  3. Use the small dropdown arrows in header cells to Filter or Sort.

Daily examples

  • Filter to show sales for Product = "Tea".
  • Sort by Date newest to oldest to see latest sales.
  • Filter by Notes contains "Online" to view only online orders.

Lesson 4 — Sorting for quick insights

Goal: find top/bottom values, latest dates or alphabetical lists quickly.

Examples to try

  • Sort by Qty descending to see top-selling items.
  • Sort by Product A → Z to group similar products together.
  • Use custom sort for multiple criteria (e.g., Product then Date).
Warning: Avoid sorting only one column — select entire table before sorting to keep rows intact.

Lesson 5 — Create clear charts (column & line)

Goal: quickly show trends or comparisons so a manager understands the story at a glance.

Step-by-step

  1. Select the two columns you want to chart (e.g., Date and Qty or Product and Qty summary).
  2. Insert → Recommended Charts → choose Column or Line.
  3. Add a short title and label axes (e.g., "Monthly Sales").
  4. Keep the chart simple — avoid too many series on one chart.
Design tip: Use a column chart for comparing products; use a line chart for trends over time.

Lesson 7 — Create a one-page Daily Sales Summary

Goal: produce a quick, printable summary a manager can review with one look.

What to include (quick checklist)

  • Date & period
  • Total sales (manually check by looking at totals column)
  • Top 3 products (sort by Qty and take top 3)
  • Small chart (column or line) showing trend
  • Notes / outstanding items (like low stock)
Mini project (do this):
  1. Use your Sales sheet and add at least 10 rows.
  2. Create a new sheet named Summary.
  3. On Summary, write: Total Sales:, Top Products: and paste a small chart.
  4. Save and export Summary as a PDF for emailing (File → Save as PDF).
Manager-ready reports are short, clear and printable. Keep the Summary to one page if you often email it.

Final practice — End-to-end

Create a workbook with two sheets: Sales and Summary. Follow these steps:

  1. Enter 12–20 sales rows in Sales sheet (use the practice table above).
  2. Format headers, currency and adjust column widths.
  3. Convert to Table (Ctrl + T) and try filtering by Product.
  4. Sort by Qty to find top-selling items and copy top 3 to Summary.
  5. Create a small chart on Summary and save as PDF.
If you are short on time — do just 6–8 rows and still create a Summary. It's about clarity, not volume.
