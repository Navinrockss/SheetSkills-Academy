Level 1 · Excel basics

Start From Zero

This level is for people who are new to Excel or feel scared to touch it. We start slow, with very small steps and real office-style examples.

Who is this course for?
  • Freshers and job seekers
  • Office staff who only open Excel but don’t know what to do next
  • Anyone who wants a very clear, simple start
Choose your learning path

You can start with Quick Path and later move to Standard or Full Path if you want more depth.

Quick Path – 3 modules

  • Module 1: Excel screen, rows, columns, cells – basic navigation.
  • Module 2: Simple numbers, SUM and AVERAGE for small tables.
  • Module 3: Save, open and basic formatting for a neat report.
Practice task:

Create a simple expense sheet for one month (date, item, amount). Use SUM to total the expenses and change the heading to bold.

Standard Path – 5 modules

  • Module 1: Workbook, sheets and moving around quickly.
  • Module 2: Entering data cleanly: dates, text, numbers.
  • Module 3: SUM, AVERAGE, MIN, MAX with small examples.
  • Module 4: Basic formatting: borders, colours, number formats.
  • Module 5: First simple chart for monthly data (like sales or expenses).
Practice task:

Make a sales sheet for 7 days with columns: Date, Product, Quantity, Rate, Amount. Use formulas to calculate Amount and create a simple column chart for daily sales.

Full Path – 7 modules

  • Module 1: Excel basics: cells, rows, columns, moving and selecting quickly.
  • Module 2: Creating simple tables with headings and proper alignment.
  • Module 3: Core formulas: SUM, AVERAGE, MIN, MAX and simple percentage.
  • Module 4: Formatting rules: number formats, bold headings, freeze top row.
  • Module 5: Small print setup: fit to one page, add title in header.
  • Module 6: Simple chart: compare two months or two products visually.
  • Module 7: Save versions, backup files, very basic good habits.
Practice task:

Create an “Attendance” sheet with Names and Days Present. Use MIN / MAX to see lowest and highest attendance and make a small bar chart.
