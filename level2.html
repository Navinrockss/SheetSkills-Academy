Level 2 · Daily office Excel

Everyday Reporting

This level is for people who already open Excel and do basic work, but want to be faster, cleaner, and more confident with real office data.

Who is this course for?
  • Back-office staff and support roles
  • Sales, HR, accounts, admin team members
  • Anyone who prepares simple reports everyday
Choose your learning path

All paths cover sorting, filtering and cleaning. Longer paths give more practice and real-life cases.

Quick Path – 3 modules

  • Module 1: Sort and filter to find records quickly.
  • Module 2: Remove duplicates and simple text cleanup.
  • Module 3: Basic conditional formatting (highlight big or small values).
Practice task:

Take a list of customers with amounts. Sort from highest to lowest, remove duplicates, and highlight customers above a target amount.

Standard Path – 5 modules

  • Module 1: Sort and filter by text, number and date.
  • Module 2: Remove duplicates, TRIM spaces, basic find/replace.
  • Module 3: Basic formulas for counts: COUNT, COUNTA, simple conditions.
  • Module 4: Conditional formatting: top 10, below average, colour scales.
  • Module 5: Small summary report from a long list (e.g. total by simple criteria).
Practice task:

Take a payment follow-up sheet. Clean duplicates, highlight overdue items, and count how many unpaid bills you have this month.

Full Path – 7 modules

  • Module 1: Sort and filter combinations (multiple columns).
  • Module 2: Data cleanup toolkit: TRIM, CLEAN, PROPER, UPPER/LOWER.
  • Module 3: Counting and basic analysis: COUNTIF examples in office.
  • Module 4: Conditional formatting for status flags and basic KPIs.
  • Module 5: Simple “Top N” list using sort and filters.
  • Module 6: Preparing a clean final sheet for sharing with your manager.
  • Module 7: Mini project: clean and summarise sales or HR data step by step.
Practice task:

Download or create a messy list (wrong cases, extra spaces, repeated names). Clean it completely and create a short summary for your boss (e.g. total sales by city).
