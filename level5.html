Level 5 · Advanced analytics

Power User Path

This level is for people who want to move closer to MIS / Business / Data Analyst roles, using tools like Power Query and simple Data Model ideas.

Who is this course for?
  • MIS and reporting professionals
  • People handling many Excel files every month
  • Learners planning to move towards analytics roles
Choose your learning path

All paths show the idea of automating repeat work. Longer paths add more steps and one full project.

Quick Path – 3 modules

  • Module 1: What is Power Query and when to use it.
  • Module 2: Import and clean one simple file with Power Query.
  • Module 3: Refresh instead of repeating steps each month.
Practice task:

Take a monthly sales CSV file. Load it into Power Query, clean a few columns, and load it back to Excel. Next month, repeat by using Refresh only.

Standard Path – 5 modules

  • Module 1: Combine multiple monthly files using Power Query.
  • Module 2: Basic transformations: remove columns, filter rows, change data types.
  • Module 3: Load cleaned data into Excel for Pivot analysis.
  • Module 4: Simple idea of related tables (master + transaction).
  • Module 5: Small “from raw to report” flow in one go.
Practice task:

Combine 3 monthly sales files (Jan, Feb, Mar) with Power Query, and then create one Pivot Table by region and month.

Full Path – 7 modules

  • Module 1: Get data from multiple files / folders in Power Query.
  • Module 2: Clean and standardise columns (names, formats, types).
  • Module 3: Load data into Data Model (where available) instead of sheet.
  • Module 4: Understand simple relationships: Customers–Sales, Products–Sales.
  • Module 5: Use Pivots from Data Model for light analysis.
  • Module 6: Refresh-friendly design: how to avoid breaking your reports.
  • Module 7: Mini project: from raw folder of files to final dashboard, refreshable every month.
Practice task:

Set up a folder with 3–4 monthly files, connect with Power Query, build a Pivot-based dashboard, and test refresh after adding another month file.
